white-space Exploration

The purpose of this is to explore how the white-space declaration works.

It impacts how white-space that occurs within the HTML document is interpreted when the webpage is painted.

It determines whether or not to pay attention to spaces, tabs, new lines that occur within the HTML document, as well as how to handle the wrapping of text.

Normal

white-space property value is normal. No matter what the text looks like in the HTML document, it will be formatted regularly, ignoring extra new lines, spaces, and tabs.

No wrap

These words are set with no-wrap white-space value. These words will break out of any limitied width containers and will jsut continue on forever in a long single line. It is not even connected to the viewport. White spaces within HTML are ignored
