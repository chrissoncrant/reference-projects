A responsive, progressivley enhanced burger menu.

Burger menus are a relic of responsive design that no matter what your opinion of them is, they continue to be a dominant design pattern. They’re very good at preserving often-limited horizontal space, but they also, more often than not, aren built in a user-hostile, non-accessible fashion.

In this premium tutorial, we’re going to build a burger menu from the ground up, using progressive enhancement, `ResizeObserver`, `Proxy` state and of course super-solid HTML and CSS that pull from the CUBE CSS principles.

This whole page is what you're building in the tutorial ☝️. Let's dive in!
